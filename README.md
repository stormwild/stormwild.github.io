stormwild.github.io
===================

Website of Alexander R. Torrijos

## Pre-requisites

- Ruby
- JekyllRb
- NodeJs
- Github Pages Gem

## Setup

Git clone this repository

```
npm install
bundle install
bundle exec jekyll serve --host $IP --port $PORT --watch --drafts
npm run start
```
